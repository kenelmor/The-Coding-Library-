# The-Coding-Library-
A library of programming books to help beginner programmers in the their preferred programming language.  
The aim for this project is to create a "library" of beginner/intermediate programming books of various programming languages. Aiming to using a Graphical User interface (which may include imaages of books) that users can view then choose if they wish to rent out book or not. May include some audio as well. Book information will stored utilzing an ArrayList. This project idea may be tweaked or changed entirely.
